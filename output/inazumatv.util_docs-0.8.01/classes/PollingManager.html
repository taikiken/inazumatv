PollingManager Class

Defined in: src/util/PollingManager.js:21
Module: inazumatv

経過時間を管理します

Example

[1] update()の戻り値で判定
make instance

 var polling1 = new PollingManager( 1000 );
 polling1.start();

check inside some loop function

 function loop () {
    requestAnimationFrame( loop );
    if ( polling1.update() ) {
      // after 1 sec.
      // do something
    }
 }

 loop();

[2] addEventListener を使う別の方法
make instance

 var polling1 = new PollingManager( 1000 );
 polling1.start();

check inside some loop function

 function eventHandler () {
      // after 1 sec.
      // do something
 }
 polling1.addEventListener( PollingManager.POLLING_PAST, eventHandler );
 function loop () {
      requestAnimationFrame( loop );
      polling1.update()
 }

 loop();
